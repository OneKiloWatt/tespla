# AUTH

## 認証方式

Supabase Auth（GoTrue）によるメール・パスワード認証。

## お試しモード

- トップ画面でアカウントを作成せずに使い始められる
- データは localStorage に保存する（キー: `tesupura-store`）
- 作成できるテスト計画は 1 つのみ
- ログインユーザーがいる場合、計画データを localStorage に書き出さない

## お試し → アカウント移行フロー

1. localStorage から現在の計画データを取得する
2. Supabase でアカウントを作成する
3. 取得した計画を DB に挿入する
4. localStorage を削除する（キー: `tesupura-store`）

## ログイン後の遷移

- ログイン成功 → ホーム（`/home`）
- 未ログイン時のアクセス制限はミドルウェア（`middleware.ts`）で制御する

## 過去の振り返り

ログインが必要。未ログインの場合はログインを促す表示にする。
